#  Calculator

Glass Calculator is a modern desktop calculator built with **Python** and **CustomTkinter**. It features a sleek dark-themed interface inspired by a glass design, making it both attractive and easy to use. The calculator performs basic arithmetic operations and provides keyboard support for faster calculations.

## Features

* Modern dark-themed user interface
* Glass-style design with rounded buttons
* Basic arithmetic operations (+, -, *, /)
* Keyboard input support
* Clear and Backspace buttons
* Hover effects on buttons
* Real-time calculation using the "=" button
* Error handling for invalid expressions

## How to Use

1. Launch the application.
2. Click the number and operator buttons or use your keyboard.
3. Press **=** or the **Enter** key to calculate the result.
4. Use **CLEAR** to remove the entire expression.
5. Use **BACKSPACE** to delete the last entered character.
